# 人生回忆录 — 部署指南

## 准备工作

1. **安装 Python**：需要 Python 3.10 或更高版本
2. **获取 DeepSeek API Key**：去 [platform.deepseek.com](https://platform.deepseek.com) 注册并获取 API Key
3. **充值**：建议先充 50-100 元，写一本回忆录（5万字）大约消耗 10-30 元

## 快速开始

### 1. 安装依赖

```bash
cd memoir-mom
pip install -r requirements.txt
```

### 2. 配置 API Key

```bash
cp .env.example .env
```

然后编辑 `.env` 文件，把 `your_deepseek_api_key_here` 换成你的真实 API Key。

### 3. 启动服务

```bash
cd backend
python main.py
```

服务启动后，在浏览器打开 `http://localhost:8000` 即可使用。

## 手机访问

如果想让妈妈在手机上用，需要让手机能访问到电脑的服务：

### 方法一：同一 WiFi（最简单）
- 确保手机和电脑在同一个 WiFi
- 启动时加 host 参数：`uvicorn main:app --host 0.0.0.0 --port 8000`
- 查看电脑 IP（`ipconfig`），然后在手机浏览器打开 `http://电脑IP:8000`

### 方法二：部署到云服务器
- 推荐 Railway / Render 等平台，一键部署
- 或者买一台最低配的国内云服务器（阿里云/腾讯云，约 50 元/月）

### 方法三：使用内网穿透（临时方案）
- 安装 ngrok 或 frp
- 生成一个公网链接，分享给妈妈

## 使用流程

1. 妈妈打开链接 → 看到三种写作风格可选
2. 选一个喜欢的（或自己描述）→ 点击"开始吧"
3. 进入聊天界面，AI 会像家人一样陪她聊人生故事
4. 随时可以：
   - 打字或用语音输入（按住 🎤 按钮说话）
   - 点「💾 存档离开」保存进度
   - 点「✨ 开始写书」让 AI 开始写
5. AI 在素材够了的时候也会主动提醒
6. 生成的章节可以阅读、修改（点"这里写得不对"）
7. 满意后导出 Word 或 PDF

## 注意事项

- 如果妈妈说"忘了/不记得了"，AI 最多追问一次就会自然跳过
- 聊天内容自动保存，关闭页面也不会丢失
- 下次打开同样的链接就能继续聊
- 建议每隔几天聊一次，每次 15-30 分钟，不用一次聊完
